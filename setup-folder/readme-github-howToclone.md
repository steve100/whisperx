# How to Download the Project

## Make or find a directory
```
   mkdir test 
   cd test
```

## Options:
   - Close with git 
   - Pull the zip file



## Use git
```
   https 
        git clone https://github.com/steve100/whisperx.git
   ssh
        git clone git@github.com:steve100/whisperx.git
```

## Use zip and unzip
```
   wget https://github.com/steve100/whisperx/archive/refs/heads/main.zip
   unzip main.zip 
```
